# ==================== 项目架构说明 ====================
# 本项目是一个Telegram菠菜投注机器人，主要功能包括：
# - 通过Telegram客户端监听群组消息
# - 使用AI模型(iFlow/阿里云)进行预测分析
# - 自动押注和结算管理
# - 多模型管理和自动降级
#
# 主要文件：
# - main.py: 主程序入口，Telegram客户端初始化
# - zq.py: 押注逻辑和状态管理
# - model_manager.py: AI模型管理器
# - variable.py: 全局变量和配置
# - config.py: 用户配置文件(需从config_example.py复制)
